============================================================
VoltageCheck: Перевірка напруги
============================================================
An operational amplifier requires a supply voltage of 15V.
The program checks an entered voltage value.
The value must be between 14.9 V and 15.1 V.
Input is repeated until the value is within the valid range.

Операційний підсилювач потребує напруги живлення 15 В.
Програма перевіряє введене значення напруги.
Значення має бути в діапазоні від 14,9 В до 15,1 В.
Введення повторюється до тих пір, поки значення не буде в допустимому діапазоні.
